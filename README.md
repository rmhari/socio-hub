# Reference Social Media Hompepage.
Social media homepage featuring posts along with their corresponding comments.

## Tech Stack
 - Node JS
 - Parcel Bundler
 - Bootstrap 5
 - Github Actions for CI
 - Github Pages for CD

## Setup
~~~ 
nvm i
npm run build
~~~
